The goal of midphys is to accompany the article about putative effect of high-level midlife
physical activity maintained till elderly life on (super)aging parameters such
as *cognitive performance*, *functional independence*, and *mental health*.

## Installation

You can install the development version of midphys from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("josefmana/midphys")
